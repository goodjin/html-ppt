# 分层架构设计

## 文档信息
- **项目名称**: Web PPT
- **版本**: v1.0
- **对应PRD**: docs/v2/01-prd.md
- **更新日期**: 2026-03-06

## 1. 层次划分

| 层次 | 名称 | 职责 | 对应PRD功能 |
|-----|------|------|------------|
| L1 | 基础设施层 | 底层能力：Vue/Vite/Reveal.js/工具库 | 所有功能 |
| L2 | 数据访问层 | 文件读写、本地存储、素材管理 | FR-001, FR-006, FR-010 |
| L3 | 业务逻辑层 | 模板服务、渲染服务、导出服务 | FR-001~FR-009 |
| L4 | 接口层 | 组件接口、事件定义、类型声明 | 所有用户故事 |
| L5 | 展示层 | 页面组件：编辑器/预览/讲者 | US-001~US-006 |

## 4. 数据流设计

### 核心数据流

```
用户输入 (Markdown/HTML)
       │
       ▼
┌──────────────────┐
│ Markdown解析     │ ← markdown-it
│ (L3: RenderService)
└──────────────────┘
       │
       ▼
┌──────────────────┐
│ HTML转换         │
└──────────────────┘
       │
       ▼
┌──────────────────┐
│ Reveal.js渲染    │ ← Reveal.js.initialize()
│ (L5: PreviewView)
└──────────────────┘
       │
       ▼
   幻灯片展示
```

### 模板管理数据流

```
用户操作 (创建/编辑/删除模板)
       │
       ▼
┌──────────────────┐
│ TemplateService  │
│ (L3)             │
└──────────────────┘
       │
       ▼
┌──────────────────┐
│ FileStorage      │ ← localfs / File API
│ (L2)             │
└──────────────────┘
       │
       ▼
   模板文件保存
```
