## 1. 系统定位

### 1.1 系统概述

Web PPT是一个基于Web技术的演示文稿开发平台，采用Vue 3 + Vite技术栈，核心依赖Reveal.js框架。系统定位为**本地优先的Web应用**，支持离线使用和CDN加载。

## 2. 技术选型

| 层次 | 技术栈 | 选型理由 | 版本要求 |
|-----|-------|---------|---------|
| 核心框架 | Vue 3 | 组合式API，与Slidev技术一致 | 3.4+ |
| 构建工具 | Vite | 快速热更新，支持Markdown | 5.0+ |
| 演示框架 | Reveal.js | 核心演示能力，成熟稳定 | 4.6+ |
| Markdown | markdown-it | 插件丰富，可扩展 | 14.0+ |
| 代码高亮 | Shiki | VS Code同款高亮，效果好 | 1.0+ |
| 状态管理 | Pinia | Vue官方推荐，轻量 | 2.0+ |
| 路由 | Vue Router | Vue官方路由 | 4.0+ |
