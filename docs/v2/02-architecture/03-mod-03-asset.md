## 3. 核心设计

### 3.1 素材结构

```typescript
interface Asset {
  id: string;              // 唯一标识
  name: string;            // 文件名
  type: AssetType;         // 文件类型
  size: number;            // 文件大小 (bytes)
  path: string;           // 存储路径
  url: string;            // 访问URL
  thumbnail?: string;     // 缩略图URL
  uploadedAt: string;     // 上传时间
}

enum AssetType {
  IMAGE = 'image',
  FONT = 'font',
  ICON = 'icon'
}
```

### 3.2 支持格式

| 类型 | 格式 | 最大大小 |
|-----|------|---------|
| 图片 | jpg, jpeg, png, gif, svg, webp | 10MB |
| 字体 | woff, woff2, ttf, otf | 5MB |

## 4. 接口定义

### 4.1 素材服务接口

```typescript
interface IAssetService {
  // 上传素材
  upload(file: File): Promise<Asset>;

  // 批量上传
  uploadMultiple(files: File[]): Promise<Asset[]>;

  // 获取素材列表
  getAssets(type?: AssetType): Promise<Asset[]>;

  // 删除素材
  delete(id: string): Promise<void>;

  // 获取素材引用路径
  getAssetUrl(id: string): string;
}
```

## 5. 边界条件

### BOUND-007: 文件类型限制

| 条件 | 处理方式 |
|-----|---------|
| 触发条件 | 上传不支持的文件类型 |
| 处理方式 | 提示"不支持的文件类型，仅支持jpg/png/gif/svg" |
| 错误码 | ASSET_TYPE_NOT_ALLOWED |

### BOUND-008: 文件大小限制

| 条件 | 处理方式 |
|-----|---------|
| 触发条件 | 上传文件超过限制大小 |
| 提示大小 | 图片10MB，字体5MB |
| 处理方式 | 提示"文件过大，请压缩后上传" |
| 错误码 | ASSET_SIZE_EXCEEDED |

### BOUND-009: 存储空间不足

| 条件 | 处理方式 |
|-----|---------|
| 触发条件 | 本地存储空间不足 |
| 处理方式 | 提示"存储空间不足，请清理不需要的素材" |
| 错误码 | ASSET_STORAGE_FULL |
