基础结构

Reveal.js 使用简单的 HTML 结构

水平幻灯片

每个 <section> 标签代表一张幻灯片

<section>幻灯片 1</section>
<section>幻灯片 2</section>

垂直幻灯片

嵌套的 section 创建垂直幻灯片

<section>
    <section>垂直幻灯片 1</section>
    <section>垂直幻灯片 2</section>
</section>
按↓键进入垂直幻灯片

代码高亮

支持多种编程语言

JavaScript

function greet(name) {
  return `Hello, ${name}!`;
}

const result = greet('World');
console.log(result);

Python

def greet(name: str) -> str:
    return f"Hello, {name}!"

result = greet("World")
print(result)

HTML

<div class="container">
    <h1>Hello World</h1>
    <p>Welcome to Reveal.js</p>
</div>

CSS

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    background: linear-gradient(45deg, #1a1a2e, #16213e);
}

JavaScript API

});
